# Timesheet Modern (corporate with life) - Full stack

## Quick start (Docker)
1. Extract ZIP and open terminal in project folder.
2. Build and run:
   docker compose build
   docker compose up
3. Open http://localhost:3000

Default accounts:
- admin / 1234
- empleado / 1234

## Run without Docker
1. Install Node.js 14+
2. Backend:
   cd backend
   npm install
   npm run init-db
   npm start
3. Open http://localhost:3000
